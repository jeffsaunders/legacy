Admin Area

The Administration Area is where admin users can create new users, configure list groups, upload groups, program settings, and generally tell SnippetMaster how it should work.

Remember to use caution when altering any options or settings in the admin area. If set improperly, some of the settings can cause confusing effects or even prevent your pages from being edited or saved properly.

Also, be very careful whom you give access to the administration area (Admin privileges). Usually, it is preferable to create regular users for your customers, instead of giving out your administrator username and password.

To access the Admin Area:

  1. Log into SnippetMaster with a user that has admin privileges.
  2. Click the admin link. (Located at the top right area of the page.)
  3. Click the tab (or link) for the admin section you want to see.

Need help?

Each option or value in the admin sections has a ? (question mark icon) beside it. If you need help with anything, just roll your mouse over (or click) the icon and a popup will appear to explain the details of the option or setting.
