How To Make A Snippet

With SnippetMaster it is possible not only to edit whole files but you can also easily define only specific portions of any page as being editable. These portions that you designate as being editable are called "Snippets".

By including Snippet Attributes when defining each Snippet, you can easily configure many different options for the Snippets. You can restrict access to only specific users, specify the size of the editor, or even disable the wysiwyg editor just for that Snippet!

Be sure to see the example files in the main program folder for a few examples on how to create different Snippet Tags and how to specify Snippet Tag Attributes.
