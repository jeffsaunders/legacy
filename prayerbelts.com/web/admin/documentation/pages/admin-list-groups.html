List Groups

A "List Group" is a group of folder names that can be used to restrict access to the filesystem of your website.

Folders can be specifically excluded or included, and you can have different lists for when the user is browsing for Snippets or files to edit, images to insert, or links on the website.

Each user can be assigned to a different List Group, so it is easy to restrict only certain users to/from certain folders.

For example, if you wanted to restrict the "jimbob"user to only have access to the "jimbobfiles" folder when navigating for Snippets, and only have access to the "images" folder when browsing for images, then here are the exact steps:

  1. Click the List Groups tab from the Admin Area.
  2. Click the New button
  3. Give the new List Group a name of "Jimbob User Only".
  4. Select Use List for the File Explorer option.
  5. Select Include for the List Type option.
  6. Enter "jimbobfiles" into the list of folder names.
  7. Select Use List for the Editor Image Browser option.
  8. Select Include for the List Type option.
  9. Enter images into the list of folder names.
  10. Click the Save button. (You have now created a new List Group.)
  11. Click on the Users tab of the Admin Area.
  12. Choose to modify the jimbob user.
  13. Scroll down to the Use This List Group option and select the "Jimbob User Only" List Group that you just created.
