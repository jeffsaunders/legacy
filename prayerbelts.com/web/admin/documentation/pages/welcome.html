Welcome

SnippetMaster is an advanced content management system (CMS) that allows non-technical users to easily update information on their websites... without having to learn HTML or fear of ruining the site design or layout.

SnippetMaster works with all major browsers, supports all CSS on your pages, and is truly "what you see is what you get".

How does it work? By adding simple "snippet" tags around any content in a web page, you can specify:

  • Exactly what content on the page can (and can't) be modified.
  • Which users can (and can't) modify certain content.
  • Who can upload files.
  • What folders certain users can see when browsing for files to edit, links or images.
  • What features of the WYSIWYG editor each user can use.
  • And much more! (See www.snippetmaster.com for full feature listing.)

Users just need to login into SnippetMaster, click on the page they want to edit, and then start typing in the WYSIWYG (What You See Is What You Get) editor. Click the save button and it's instantly updated! What could be easier?

If you prefer to edit an entire page, then this is easily possible! (Users with Admin or Super User privileges are not limited only to snippets.)

No matter what kind of site you have, SnippetMaster won't take long at all to setup. And with SnippetMaster you will always be sure your website is working properly - no matter how many changes your clients make.

  • Reduce website management costs
