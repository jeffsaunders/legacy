Adding Users

Adding users to SnippetMaster is easy.

  1. Log into SnippetMaster with a user that has admin privileges.
  2. Click the admin link. (Located at the top right area of the page.)
  3. Click the Users tab.

Each user can be assigned unique privileges and options. It is possible to:

  • Allow only certain users to have access to certain snippets. (See details about the "users snippet tag attribute" here.)
  • Assign a user to any List Group so you can restrict what folders they can and can not "see" when browsing for snippets, images, and links.
  • Assign a user to any Upload Group so you can specify exactly what folders they can upload files into.
  • Assign a user to any Editor Group so you can hide certain features of the WYSIWYG editor.
  • Enable or disable the ability to use the File Upload Utility.
  • Give a user the ability to edit -all- snippets in any file, no matter if they are listed in the "users attribute" for that snippet or not. (Power User privilege)
  • Give any user access to the admin area (Admin User privilege)
  • Set the language for each user, individually.
