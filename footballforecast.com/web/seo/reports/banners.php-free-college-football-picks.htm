Do the following to improve the ranking for your web site on Google.com for the search term "free college football picks"

 

General advice about inbound links: Many search engines think that your web page must be relevant for the search term "free college football picks" if other web pages link to your page with "free college football picks" in the link text.


If many web pages link to your page with the link text "free college football picks", then it's more likely that your web page will obtain a good ranking for the search term "free college football picks".

 

How inbound links to your site should look like: When you exchange links with other web sites, make sure that your important keywords are in the link texts.


For example, if 20 other web sites link to your page with the HTML source 'Click here for <a href="http://www.footballforecast.com/banners.php">free college football picks</a>', then it's much more likely that you'll obtain a good ranking for the search term "free college football picks" than if 100 web sites link to your page with '<a href="http://www.footballforecast.com/banners.php">Click here for more information</a>.'
